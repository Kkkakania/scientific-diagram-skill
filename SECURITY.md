# Security

This repository should not receive private diagrams, datasets, screenshots,
emails, local paths, or hidden source material.

If you find a privacy or provenance problem in a committed file, do not open a
public issue with the sensitive content. Open a short issue that says which
public file appears risky, or contact the maintainer through the GitHub profile.
Do not paste the private value itself.

Security-relevant examples:

- a committed local path or email address;
- a `.drawio` file that appears copied from a private source;
- an SVG export with external links, script content, or embedded private data;
- a public example that appears to trace a paper figure or screenshot.

For ordinary documentation fixes, use the normal issue or pull request flow.
