# Code of Conduct

Be direct, kind, and specific. Review ideas and reproducible behavior without
attacking the person who proposed them.

Issues and pull requests should stay focused on scientific-diagram structure,
documentation, accessibility, and clean-room provenance. Harassment, personal
attacks, discriminatory language, repeated disruption, and attempts to publish
private or identifying material are not acceptable.

Maintainers may edit or remove comments, close contributions, or restrict
participation when this standard is not met. Report ordinary conduct concerns to
the maintainers. When a public report would expose private material, use the
private reporting path in [SECURITY.md](SECURITY.md).
